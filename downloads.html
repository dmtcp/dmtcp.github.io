Getting DMTCP:

DMTCP version 2.3, released on July 03, 2014, is now available. Binary packages for Debian, Fedora, and OpenSUSE would be submitted over the next few days.

The latest stable version of DMTCP (along with earlier releases) can always be obtained through the sourceforge.net downloads page. Newer users are recommended to download DMTCP version 2.x. (Version 1.x is maintained for backward compatibility.)

  • DMTCP version 2.x
     
  • DMTCP version 1.x
     

To obtain the most recent (possibly unstable) source from the subversion repository, run the following command:

  • READ-ONLY: 
     svn co svn://svn.code.sf.net/p/dmtcp/code/trunk dmtcp-trunk
  • READ-WRITE (developers only, replace USER with username): 
     svn co svn+ssh://USER@svn.code.sf.net/p/dmtcp/code/trunk dmtcp-trunk
  • OBSOLETE (lmited to older revisions): 
     svn co https://dmtcp.svn.sourceforge.net/svnroot/dmtcp/trunk dmtcp
  • or else download a zip file of the latest svn revision from:
    http://sourceforge.net/p/dmtcp/code/HEAD/tarball?path=/trunk

  • BROWSE or DOWNLOAD a revision from the svn (experts only):
    Go to the "code" tab at sourceforge.
    Then select "trunk", "tags", or other, and either browse the code directly or click on "Download Snapshot".
