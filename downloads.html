Getting DMTCP:

DMTCP version 2.3.1, released on July 14, 2014, is now available. Binary packages for Debian, Fedora, and OpenSUSE would be submitted over the next few days.

The latest stable version of DMTCP (along with earlier releases) can always be obtained through the sourceforge.net downloads page. Newer users are recommended to download DMTCP version 2.x. (Version 1.x is maintained for backward compatibility.)

  • DMTCP version 2.x
     
  • DMTCP version 1.x
     

  • To obtain the most recent (possibly unstable) source from the github repository, run the following command: 
     git clone https://github.com/dmtcp/dmtcp.git
  • or else download a zip file of the latest svn revision from:
    https://github.com/dmtcp/dmtcp/archive/master.zip
