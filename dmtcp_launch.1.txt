[1mNAME[0m
       [1mdmtcp_launch [22m-- start a process under DMTCP control.

[1mSYNOPSIS[0m
       [1mdmtcp_launch [22m[[4mOPTIONS[24m] [4m<command>[24m [[4margs...[24m]

[1mDESCRIPTION[0m
       [1mdmtcp_command  [22mis a tool to send user commands to the [1mdmtcp_coordinator[0m
       remotely.

[1mOPTIONS[0m
   [1mConnecting to the DMTCP Coordinator[0m
       [1m-h[22m, [1m--coord-host [4m[22mhostname[24m (environment variable DMTCP_COORD_HOST)
               Hostname where dmtcp_coordinator is run (default: localhost)

       [1m-p[22m, [1m--coord-port [4m[22mport[24m (environment variable DMTCP_COORD_PORT)
               Port where dmtcp_coordinator is run (default: 7779)

       [1m--port-file [4m[22mfilename[0m
               File to write listener port number.  (Useful with  [1m--coord-port[0m
              [1m0[22m, which is used to assign a random port)

       [1m-j[22m, [1m--join[0m
               Join  an  existing  coordinator,  raise  error  if  one doesn't
              already exist

       [1m--new-coordinator[0m
               Create a new coordinator at the given port. Fail if one already
              exists  on  the  given  port.  The  port  can  be specified with
              [1m--coord-port[22m, or with environment variable DMTCP_COORD_PORT.  If
              no  port  is specified, start coordinator at a random port (same
              as specifying port '0').

       [1m--no-coordinator[0m
               Execute the process in stand-alone coordinator-less mode.
              Use [1mdmtcp_command [22mor [1m--interval [22mto request checkpoints.

       [1m-i[22m, [1m-interval [4m[22mseconds[24m (environment variable DMTCP_CHECKPOINT_INTERVAL)
               Time in seconds between automatic checkpoints.  0 implies never
              (manual ckpt only); if not set and no env var, use default value
              set in  dmtcp_coordinator  or  dmtcp_command.   Not  allowed  if
              [1m--join [22mis specified

   [1mCheckpoint image generation[0m
       [1m--gzip[22m, [1m--no-gzip [22m(environment variable DMTCP_GZIP=[01])
               Enable/disable  compression  of  checkpoint  images (default: 1
              (enabled))
              WARNING: gzip adds seconds. Without gzip, ckpt is often < 1s

       [1m--ckptdir [4m[22mpath[24m (environment variable DMTCP_CHECKPOINT_DIR)
               Directory to store checkpoint  images  (default:  curr  dir  at
              launch)

       [1m--ckpt-open-files[0m
               Checkpoint open files and restore old working dir. (default: do
              neither)

       [1m--checkpoint-open-files[0m
               Deprecated. Use [1m--ckpt-open-files [22minstead.

       [1m--ckpt-signal [4m[22msignum[0m
               Signal  number  used  internally  by  DMTCP  for  checkpointing
              (default: 12)

       [1m--ckpt-signal [4m[22msignum[0m
               Deprecated. Use [1m--ckpt-signal [22minstead.

   [1mEnable/disable plugins[0m
       [1m--with-plugin [4m[22mplugins[24m (environment variable DMTCP_PLUGIN)
               Colon-separated  list  of  DMTCP  plugins  to be preloaded with
              DMTCP.  (Absolute pathnames are required.)

       [1m--batch-queue[22m, [1m--rm[0m
               Enable support for resource managers (Torque  PBS  and  SLURM).
              (default: disabled)

       [1m--ptrace[0m
               Enable  support  for  PTRACE  system  call  for gdb/strace etc.
              (default: disabled)

       [1m--modify-env[0m
               Update environment variables based on the  environment  on  the
              restart host (e.g., DISPLAY=$DISPLAY). This can be set in a file
              dmtcp_env.txt.  (default: disabled)

       [1m--ib[22m, [1m--infiniband[0m
               Enable InfiniBand plugin. (default: disabled)

       [1m--disable-alloc-plugin [22m(environment variable DMTCP_ALLOC_PLUGIN=[01])
               Disable alloc plugin (default: enabled).

       [1m--disable-dl-plugin [22m(environment variable DMTCP_DL_PLUGIN=[01])
               Disable dl plugin (default: enabled).

       [1m--disable-all-plugins [22m(EXPERTS ONLY, FOR DEBUGGING)
               Disable all plugins.

   [1mOther options[0m
       [1m--prefix [4m[22mpath[0m
               Prefix where DMTCP is installed on remote nodes.

       [1m--tmpdir [4m[22mpath[24m (environment variable DMTCP_TMPDIR)
               Directory   to   store   temporary   files   (default:    $TMD‐
              PIR/dmtcp-$USER@$HOST or /tmp/dmtcp-$USER@$HOST)

       [1m-q[22m, [1m--quiet [22m(or set environment variable DMTCP_QUIET = 0, 1, or 2)
               Skip NOTE messages; if given twice, also skip WARNINGs

       [1m--help[0m
               Print this message and exit.

       [1m--version[0m
               Print version information and exit.
